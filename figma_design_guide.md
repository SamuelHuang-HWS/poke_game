# 炸金花游戏平台 - Figma界面设计指南

## 设计概述
基于一期核心功能需求，设计具有科技未来感的炸金花游戏界面，采用深色主题配合霓虹光效，营造高端专业的游戏氛围。

## 色彩方案

### 主色调
- **背景深空蓝**: #1a1a2e
- **面板深蓝**: #16213e
- **霓虹紫**: #6c5ce7
- **电光绿**: #00d2d3
- **警告橙**: #ff7675

### 辅助色彩
- **成功绿**: #00b894
- **信息蓝**: #74b9ff
- **文字银**: #f0f0f0
- **文字灰**: #b2b2b2

## 核心界面设计

### 1. 登录/注册界面
**布局结构**:
- 居中卡片式布局
- 半透明玻璃拟态效果
- 霓虹光边框装饰

**元素设计**:
- Logo区域：发光文字效果
- 手机号输入框：圆角矩形，聚焦时紫色光晕
- 验证码输入：简洁数字输入框
- 登录按钮：渐变紫色背景，悬停发光
- 背景：动态粒子效果

### 2. 主界面（房间中心）
**布局结构**:
- 顶部用户信息栏
- 中央功能按钮区域
- 底部导航栏

**核心元素**:
```

├─────────────────────────────────────┤
```

**视觉特效**:
- 按钮：3D悬浮效果，点击凹陷动画
- 背景：流动的数据流动画
- 边框：呼吸灯效果

### 3. 创建房间界面
**表单设计**:
- 房间名称输入框
- 入场金币选择器（滑块或下拉）
- 单注金额选择器
- 局数设定（下拉选择：10/20/30局）
- 房间密码（可选）

**交互设计**:
- 实时显示房间配置预览
- 金币范围验证提示
- 创建成功后的房间号展示动画

**牌型图标**:

- 同花：三朵花形图案
- 顺子：连续的数字阶梯
- 对子：两个相连的圆点
- 单张：单独的扑克牌图标

**动画效果**:
- 牌型出现时的粒子爆炸效果
- 胜利时的金币雨动画
- 失败时的淡出效果

## 交互动画设计

### 页面转场
- 滑动切换：左右滑动切换主要界面
- 淡入淡出：模态框的出现和消失
- 缩放动画：按钮点击的反馈

### 游戏内动画
- 发牌动画：扑克牌从中央飞向玩家
- 翻牌动画：3D翻转效果
- 金币移动：下注时的金币流动动画
- 比牌动画：两张牌靠近对比

### 状态反馈
- 加载状态：旋转的加载图标
- 成功状态：绿色勾选配合粒子效果
- 错误状态：红色抖动配合警告图标
- 网络状态：连接中断时的提示动画

## 响应式设计

### 移动端适配
- 竖屏模式：优化单手操作
- 横屏模式：最大化游戏视野
- 不同尺寸：自适应布局和字体大小

### 平板适配
- 横屏双面板：左侧游戏区，右侧聊天区
- 竖屏单面板：全屏游戏界面

## 可访问性设计

### 视觉辅助
- 高对比度模式
- 色盲友好的色彩方案
- 字体大小调节选项

### 操作辅助
- 大按钮模式
- 语音播报功能
- 手势操作支持

## 设计交付规范

### 切图规范
- SVG格式：图标和简单图形
- PNG格式：复杂图像和照片
- 多倍图：@1x, @2x, @3x

### 标注规范
- 间距标注：8px网格系统
- 字体规范：字号、字重、行高
- 颜色标注：十六进制和RGB值

### 动效规范
- 时长标准：200ms快速反馈，400ms页面转场
- 缓动函数：ease-in-out为主
- 性能考虑：GPU加速优先

## 技术实现建议

### CSS实现
```css
/* 玻璃拟态效果 */
.glass-effect {
  background: rgba(255, 255, 255, 0.1);
  backdrop-filter: blur(10px);
  border: 1px solid rgba(255, 255, 255, 0.2);
}

/* 霓虹发光效果 */
.neon-glow {
  box-shadow: 0 0 20px rgba(108, 92, 231, 0.5);
  animation: neon-pulse 2s infinite;
}

@keyframes neon-pulse {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.7; }
}
```

### 动画实现
- 使用CSS3动画处理简单动效
- 复杂动画使用Lottie或GSAP
- 游戏动画使用Canvas或WebGL
