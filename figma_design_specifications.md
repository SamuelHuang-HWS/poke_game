## 4. 动画规范

### 4.1 基础动画时长
```
快速反馈: 150-200ms
├── 按钮点击
├── 状态切换
└── 悬停效果

标准转场: 300-400ms
├── 页面切换
├── 模态框
└── 卡片展开

复杂动画: 600-800ms
├── 发牌动画
├── 金币流动
└── 结算展示
```

### 4.2 缓动函数
```
标准缓动: cubic-bezier(0.4, 0, 0.2, 1)
加速缓动: cubic-bezier(0.4, 0, 1, 1)
减速缓动: cubic-bezier(0, 0, 0.2, 1)
弹性缓动: cubic-bezier(0.68, -0.55, 0.265, 1.55)
```

### 4.3 关键动画效果
```
霓虹发光:
@keyframes neon-glow {
  0%, 100% { 
    box-shadow: 0 0 5px #6c5ce7,
                0 0 10px #6c5ce7,
                0 0 15px #6c5ce7; 
  }
  50% { 
    box-shadow: 0 0 10px #6c5ce7,
                0 0 20px #6c5ce7,
                0 0 30px #6c5ce7; 
  }
}

呼吸灯效果:
@keyframes breathing {
  0%, 100% { opacity: 0.6; }
  50% { opacity: 1; }
}

```
